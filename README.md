# LibPlcTag.FSharp

**LibPlcTag.FSharp** is a thin F# wrapper around [libplctag](https://github.com/libplctag/libplctag), the C library for PLC communication.

```bash
dotnet add package LibPlcTag.FSharp --version 2.2.2
```
